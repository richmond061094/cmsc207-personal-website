# CMSC 207 Mini-Project: Personal Web Profile

## Project Overview
This repository contains my mini-project for CMSC 207. It is a single-page personal web profile built to demonstrate foundational web development concepts, inspired by the principles from *Create with Code: Build Your Own Website* by Clyde Hatter.

**Author:** Richmond Manuel  
**Live Demo (Vercel):** https://cmsc207-personal-website.vercel.app/

## Project Objectives & Features
This project applies the following core web development skills:
- **Semantic HTML5:** Structured using proper tags (`<header>`, `<nav>`, `<section>`, `<footer>`).
- **CSS Styling & Layout:** Implements external stylesheets for a clean, visually appealing design, including hover effects on navigation links, images, and buttons.
- **JavaScript Interactivity:** Features a dynamic interactive element (a button that reveals a hidden fun fact about my background).
- **Responsive Design:** Utilizes CSS Flexbox, Grid, and media queries to ensure the site is fully functional and visually consistent across both desktop and mobile devices.

## Sections Included
1. **Header:** Site title and working navigation menu with smooth scrolling.
2. **About Me:** A brief professional introduction and profile image.
3. **Hobbies & Interests:** A grid layout detailing my interests in running, continuous learning, and GIS/Web Automation.
4. **Contact Info:** Reachable links for email and professional profiles.

## Technologies Used
* **HTML5**
* **CSS3** (Custom properties, Flexbox, Grid, Media Queries)
* **Vanilla JavaScript** (DOM manipulation and Event Listeners)

## Setup and Installation
To view this project locally:
1. Clone the repository: `git clone https://github.com/richmond061094/cmsc207-personal-website.git`
2. Open the project folder in your preferred code editor (e.g., VS Code).
3. Run the `index.html` file in your browser or use an extension like Live Server.
